# Creators — practice

Figure sources for the *Chaos in practice* chapter (`maple/practice.md`; figures
live in `maple/_static/practice/`).

Unlike the rest of the book, whose figures come from Maple worksheets, the
figures of this chapter are generated by a single **Python** script,
[`make_figures.py`](make_figures.py). This is deliberate: the chapter is
forward-looking and its examples (ensemble forecasting, the Kalman filter,
POD/Koopman) are most naturally expressed with `numpy`/`matplotlib`, and it
doubles as a first Python creator ahead of the planned Python edition (see
`ROADMAP.md`). The *code shown in the book itself remains Maple*, matching the
Maple edition; the Python here only draws the figures.

## Reproduce

```
python creators/practice/make_figures.py
```

Requires `numpy`, `scipy` and `matplotlib`. All eight PNGs are written to
`maple/_static/practice/`.

## Notes on the computed figures

- **Lorenz system** with the book's parameters `sigma=10, r=28, b=8/3`, integrated
  with a fixed-step RK4 (`make_figures.py:rk4_step`). Initial conditions are placed
  on the attractor by discarding a transient (`on_attractor`), exactly as in
  chapter `cont3d`.
- **Ensemble / horizon.** `M` members from a Gaussian blob of width `eps0`; the
  spread is the RMS distance from the ensemble mean. The growth rate matches the
  effective Lyapunov exponent `Lambda_e ≈ 0.9` computed in chapter `cont3d`.
- **Assimilation.** An extended Kalman filter whose covariance is propagated with
  the Lorenz Jacobian (`jacobian`) — the same `J` used for the Lyapunov exponents
  in `cont3d`. The full state is observed every `dt_obs = 0.2` with noise `sigma=1.5`;
  mild multiplicative inflation (1.03) keeps the small-ensemble covariance healthy.
- **POD.** A synthetic field `u(x,t) = sum_k a_k(t) phi_k(x) + noise` whose modal
  amplitudes `a_k(t)` are a normalised Lorenz trajectory, so the reduction ties back
  to the attractor of `cont3d`. The SVD of the snapshot matrix gives the spectrum.
