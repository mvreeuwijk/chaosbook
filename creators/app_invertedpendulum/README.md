# Creators — app_invertedpendulum

Figure sources for the *Stability of the inverted pendulum* appendix
(`latexsource/app_invertedpendulum.tex`; figures live in
`latexsource/Figs_app_invertedpendulum/`).

- **`app_invpend_stability.mw`** — Maple worksheet that produces
  `app_invpend_stability.eps`, the $(\Omega,\alpha)$ stability diagram of the
  vibrated pendulum. Copied from `latexsource/exdb/ex0111r00_stability.mw`.
- `app_invpend_sketch.eps` — hand-drawn definition sketch of the pendulum with a
  vibrating suspension point. **No Maple/vector source** exists in the repo
  (originally a drawn/xfig figure); the `.eps` is the master.
